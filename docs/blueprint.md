## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism.
- Background color: Light gray (#F0F2F5), subtly desaturated from the primary.
- Accent color: Purple (#8E24AA) for highlighting key elements.
- Body text and headline font: 'Inter', a grotesque-style sans-serif.
- Use simple, outline-style icons to maintain a clean and modern aesthetic.
- Employ a clean and grid-based layout to showcase projects and information in a structured manner.
- Incorporate subtle transition animations when navigating between sections to enhance user experience.
